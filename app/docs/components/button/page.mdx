## Usage

```tsx
import { Button } from "@/components/ui/button"

export function Example() {
  return <Button>Continue</Button>
}
```

## Variants

The component ships with six variants:

- `default`
- `secondary`
- `outline`
- `ghost`
- `destructive`
- `link`
