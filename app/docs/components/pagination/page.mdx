## Usage

```tsx
import {
  Pagination,
  PaginationButton,
  PaginationGap,
} from "@/components/ui/pagination"

export function Example() {
  return (
    <Pagination>
      <PaginationButton disabled>Previous</PaginationButton>
      <PaginationButton active>1</PaginationButton>
      <PaginationButton>2</PaginationButton>
      <PaginationGap>...</PaginationGap>
      <PaginationButton>8</PaginationButton>
      <PaginationButton>Next</PaginationButton>
    </Pagination>
  )
}
```

## Notes

- `Pagination` is intentionally unopinionated about routing; wire it to links or handlers as needed.
- Use `active` to promote the current page visually.
