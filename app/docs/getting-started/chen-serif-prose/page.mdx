## When to Use It

- Use `chen-serif-prose` when your project already has its own components but you want Chen Serif's editorial markdown treatment.
- Use `chen-serif` instead when you want the full design system package, including tokens, global utilities, and the core UI components.

## What You Get

- Heading, paragraph, link, list, inline code, blockquote, and table styles through the `.chen-serif-prose` class.
- Dependency on `chen-serif-theme`, so the required font aliases and design tokens are installed automatically.

## Usage

```tsx
export function ArticleBody() {
  return (
    <article className="chen-serif-prose">
      <h1>Editorial heading</h1>
      <p>
        This paragraph picks up the Chen Serif prose typography.
      </p>
    </article>
  )
}
```
